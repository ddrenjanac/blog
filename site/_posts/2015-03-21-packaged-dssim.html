I just packaged dssim for ubuntu. So for now it can easily installed and removed as plain deb file.

To install just type:
{% highlight bash %} $ sudo apt-add-repository ppa:lkwg82/dssim $ sudo apt-get update $ sudo apt-get install dssim {% endhighlight %}
Currently it is only available for ubuntu trusty (14.04 LTS). On request I'll make it for other versions as well.
