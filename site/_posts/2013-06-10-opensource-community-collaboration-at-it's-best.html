--- layout: post title: opensource community collaboration at it's best date: 2013-06-10 18:56:00 UTC updated: 2013-06-10 18:56:00 UTC comments: true categories: --- Today I had a great experience with contributing to an opensource project redline (Redline is a pure Java library for manipulating RPM Package Manager packages). Actually it was last week and today I had this pleasure.

Last week I fixed some lines in code and pulled a merge request. The same day, just 2 hours later, it was merged.

Today I asked for a release to benefit from these changes without publishing an separate inhouse version. I offered some additional refactorings to justify a release, however Craig W. Blake did not discussed or requested some more changes. An new release is already online.

I really appreciate that kind of cooperation. That's why I love opensource and my company Immobilienscout24 is sharing and contributing to the opensource world.

See our projects at github.com/ImmobilienScout24. Feel free to use it, to contribute or just report bugs!
