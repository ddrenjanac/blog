I wanna use some resources on my classpath and dont know the exact path to address them. I'll show this using an example with jquery.

I have dependency and want to use the jquery inside. {% highlight xml %} ... ... org.webjars jquery 2.1.4 ... com.google.guava guava 18.0 ... {% endhighlight %}

With this demo test: {% highlight java %} import com.google.common.reflect.ClassPath; import org.junit.Test; import java.io.IOException; import java.net.URL; import java.nio.file.Files; import java.nio.file.Paths; public class DemoTest { private static String readAsString(String name) throws IOException { URL resource = ResourceUtils.class.getClassLoader().getResource(name); if (resource == null) { throw new IOException("resource not found:" + name); } String path = resource.getFile(); return new String(Files.readAllBytes(Paths.get(path))); } @Test public void test() throws Exception { ClassPath.from(getClass().getClassLoader()) .getResources() .stream() .filter(e -> e.getResourceName().contains("jquery.min.js")) .forEach(e-> { System.out.println("path : " + e); System.out.println("first bytes: " + readAsString(e.getResourceName()).substring(0,10)); }); } } {% endhighlight %}

The output is: {% highlight java %} path : META-INF/resources/webjars/jquery/2.1.4/jquery.min.js first bytes: /*! jQuery {% endhighlight %}
